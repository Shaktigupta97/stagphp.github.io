StagPHP Framework

Cyzer is the very lightweight PHP framework. It is compatible with almost all web server as it required PHP, my SQL and Apache to run. A developer can choose the type of controllers for their application these controllers are the main engine fo the application. Cyzer comes with multiple controllers other than default controllers and modules additional controllers can also be added as an add-on or plugin.

  • Fully Open Source

    Every piece of code written to build cyzer resides on Github public repositories. Fork Cyzer on Github.

  • Controlled Development Environment

    Cyzer do not load any controllers by default. You can select the controllers and include it in your application as per your requirement.

  • Extensible and Secure

    You can extend the functionality of Cyzer by using controller modules as a addon. Installing trusted controller modules will keeps your Cyzer application safe and secure with regular updates.
